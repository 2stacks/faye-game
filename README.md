# Faye Game

A collection of browser-based games built with AI-assisted coding to show an 11-year-old video game fan how easy and fun it is to create games with AI.

Every game is a single HTML file — no installs, no build tools, just open and play.

## Games

- **Flappy Boom** (`flappy.html`) — A Flappy Bird clone featuring Boom the golden retriever, with scrolling pipes, score tracking, and high score persistence.

## How to Play

Open any game file in your browser:

```bash
xdg-open flappy.html
```

Or just double-click the `.html` file.

**Controls:** Click, tap, Spacebar, or Arrow Up to flap.

## Built With

- HTML5 Canvas
- Vanilla JavaScript
- Claude Code (AI pair programming)
